# ① node v22.x 下で yarn install & yarn web

```
$ node -v
v22.x

$ yarn install
$ yarn web
```

↓↓↓↓<br>

Open <a href="http://localhost:8081" target="_blank">http://localhost:8081</a> in your browser<br>
<br>
